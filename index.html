Sistema de Irrigação

Projeto pessoal de irrigação automatizada para aprendizado e resolução de um problema familiar, usando MySQL, PHP, JavaScript, JAVA e C++

O que o sistema faz?

O sistema faz a leitura de 3 sensores (temperatura ambiente, umidade relativa do ar e umidade do solo) a cada 1 minuto (esse valor pode ser modificado), depois salva os dados das leituras no banco de dados, se o valor de umidade do solo estiver abaixo do valor limiar (esse valor pode ser modificado) ele irá ativar o sistema de irrigação automaticamente, depois disso ele salva os dados das leituras no banco de dados, podendo ser visualizado na aba de controle de sensores, onde podemos ver os dados dos sensores e ativar o sistema de irrigação manualmente. Nessa mesma página podemos ver o gráfico com todas as leituras e se for necessário, imprimir o mesmo. Podemos consultar todas as leituras, por dia/mês/ano ou por mês/ano e gerar o relatório do dia.
